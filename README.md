# Analysis-of-Natural-Radionuclides-in-Quartz-Sand-Samples-Using-a-Gamma-Spectrometer
Repositori ini berisi laporan penelitian mengenai analisis kandungan radionuklida alam pada sampel pasir kuarsa dari wilayah bekas penambangan timah di Kepulauan Bangka Belitung. Penelitian ini dilakukan menggunakan metode spektrometri gamma dengan bantuan perangkat lunak Y-Spect untuk mengolah dan menganalisis data hasil pengukuran.

📖 Latar Belakang

Wilayah bekas penambangan timah memiliki potensi kandungan radionuklida alam akibat proses geologi dan aktivitas pertambangan. Oleh karena itu, analisis radioaktivitas lingkungan menjadi penting untuk mengetahui tingkat paparan radiasi.

Radionuklida yang umum dianalisis meliputi:

Uranium (U-238)
Thorium (Th-232)
Kalium (K-40)

🧠 Dasar Teori

Penelitian ini didasarkan pada konsep:

Fisika Nuklir
Radioaktivitas
Spektrometri Gamma

Spektrum energi gamma yang diperoleh digunakan untuk mengidentifikasi jenis radionuklida dan menghitung aktivitasnya.

⚙️ Metodologi

Langkah-langkah penelitian:

Pengambilan sampel pasir kuarsa dari lokasi bekas tambang
Preparasi sampel (pengeringan dan homogenisasi)
Pengukuran menggunakan spektrometer gamma
Akuisisi data spektrum
Analisis spektrum menggunakan Y-Spect
Identifikasi puncak energi radionuklida
Perhitungan aktivitas (Bq/kg)

💻 Perangkat Lunak

Analisis data dilakukan menggunakan:

Y-Spect

Fungsi utama Y-Spect dalam penelitian ini:

Menampilkan spektrum energi gamma
Identifikasi puncak (peak detection)
Kalibrasi energi
Perhitungan luas puncak
Penentuan aktivitas radionuklida

🔬 Alat dan Bahan

Spektrometer gamma 
Sampel pasir kuarsa
Wadah sampel
Komputer/laptop dengan Y-Spect

📊 Hasil yang Diharapkan

Spektrum energi gamma dari sampel
Identifikasi radionuklida (U-238, Th-232, K-40)
Nilai aktivitas radionuklida (Bq/kg)
Analisis tingkat radioaktivitas lingkungan

🎯 Tujuan

Menentukan kandungan radionuklida alam pada pasir kuarsa
Menggunakan Y-Spect untuk analisis spektrum gamma
Mengevaluasi potensi bahaya radiasi di area bekas tambang
